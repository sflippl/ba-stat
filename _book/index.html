English

During data analysis, statistician have to explicitly notate certain parameters many times. For instance, a scale transformation would have to be specified for every plot which includes this variable. This problem is aggravated during the stage of presentation; as these decisions are not explicitly denoted, it is time expensive to provide

The statistical programming language R provides powerful opportunities to outsource everyday thought processes in data analysis. However, adapting these mechanisms for application-specific thought processes is expensive and difficult. Such an adaptation would have a rich field of applications, in particular within the social sciences. In my Bachelor’s thesis, I propose a framework to specify default visualizations and descriptions for different variables in an explicit and effective way. I implement the framework in the R package tectr and conduct a case study with the Varieties of Democracy database.
